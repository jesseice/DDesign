## API

| 名称 |  类型  |  默认值   | 说明                                                  | 必传 |
| :--: | :----: | :-------: | ----------------------------------------------------- | ---- |
| type | string | `default` | 类型 `default` `primary` `success` `warning` `danger` | N    |
| size | string | `default` | 尺寸 `small` `default` `large`                        | N    |
